# 📌 Sistema de Gerenciamento de Hóspedes

## 🏨 Sobre o Projeto

Este sistema tem como objetivo facilitar a gestão de hóspedes em um hotel, permitindo check-in, check-out, integração de dados e visualização de informações através de um dashboard intuitivo.

## 📂 Estrutura do Projeto

O projeto está dividido em quatro equipes, cada uma responsável por uma parte essencial do sistema:

```
📦 hotel_management
├── 📁 interface_dashboard
│   ├── dashboard.py
│
├── 📁 checkin
│   ├── checkin.py
│
├── 📁 integracao
│
├── 📁 checkout
│   ├── checkout.py
│   ├── utils.py
│
├── 📁 database
│   ├── dataBase.txt
│   └── historico.txt

├── .gitignore
├── README.md
└── main.py
```

## 🚀 Como Executar o Projeto

### 🔧 Pré-requisitos

Certifique-se de ter o **Python 3.x** instalado.

1. Clone este repositório:
   ```bash
   git clone https://github.com/KeylaOliver/CapacitaM2LP
   ```
2. Acesse a pasta do projeto:
   ```bash
   cd CapacitaM2LP
   ```
3. Execute o sistema::
   ```bash
   python main.py
   ```

Desenvolvido pelas equipes do projeto! 💻✨
